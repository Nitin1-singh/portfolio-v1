## Getting Started

To duplicate my work, follow the guidelines listed below.

### Prerequisites

The list of software you need:

* Latest Version of Node
* Latest Version of Git
* An account on Brevo for email api

### Installation

1. Get a free API Key at [brevo.com](https://app.brevo.com/)
2. Clone the repo
   ```sh
   git clone https://github.com/Nitin1-singh/portfolio-v1.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API KEY in .env
   ```js
   EMAIL_API = 'ENTER YOUR API KEY';
   ```
